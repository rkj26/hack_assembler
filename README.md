# Hack Assembler

## Context
This repository builds the Hack Assembler that is part of the Nand2Tetris Part I Course. 

You can find more about the course here: https://www.nand2tetris.org
The work of this repository deals with Ch6 - Assembler of the Nand2Tetris Part I Course.

## Aim
Aim of this project is to get used to:
- Modern C++ along with CMake.
- Build the Assembler in a clean and organised manner for the Nand2Tetris Part I Course.

## Instructions
How to run it for the first time
```
> mkdir build && cd build && cmake ../CMakeLists.txt
> cmake --build .
> ./HackAssembler
```

Subsequent runs can be done as follows:
```
> cd build && cmake ..
> cmake --build .
> ./HackAssembler
```
